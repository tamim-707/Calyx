# Calyx AI Assistant

Calyx is a Python-based personal AI assistant built by Tamim.
Version: v1.0

## Features
- Greeting system
- Calculator
- Time & Date
- Memory system
- Notes system
- Personality modes
- News API
- Weather API
- Chat history
- App launcher

## Commands
- hello
- help
- time
- date
- calc
- history
- weather of dhaka
- news of sports in us
- change to funny

## Installation
Clone the repository:

```bash
git clone https://github.com/tamim-707/Calyx.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Calyx:

```bash
python Calyx.py
```

## Tech Used
- Python
- Requests API
- JSON storage
- Modular architecture
